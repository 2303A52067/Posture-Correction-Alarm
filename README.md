
# Posture Correction Alarm using TensorFlow.js

## Overview
This project is a smart posture correction system that detects improper posture and alerts the user in real-time using a warning sound and visual feedback.

## Objective
To help individuals avoid back pain and posture-related health issues caused by prolonged sitting.

## Features
* Real-time posture detection
* Warning sound alert
* Screen blur for incorrect posture

## Technologies Used
* TensorFlow.js
* HTML, CSS, JavaScript

## Working
The system uses TensorFlow.js to monitor user posture via the webcam.
If incorrect posture is detected:
→ Warning sound is triggered
→ Screen gets blurred

## Project Structure
* `index.html` – Main interface
* `style.css` – Styling
* `video.js` – Logic & detection
* `assets/` – Audio and fonts
* `demo/` – Demo video
